# ECommerce Analysis Excel Dashboard
This Excel Capstone Project delivers an end‑to‑end E‑Commerce Sales Analysis Dashboard that combines thorough data cleaning with advanced formula‑based calculations to provide clear business insights. Raw sales data was standardized by removing duplicates, handling missing values, and formatting fields for consistency, while formulas such as SUMIFS, VLOOKUP, and calculated averages were applied to derive category‑wise totals, payment type distributions, regional comparisons, and loyalty segmentation. The final dashboard integrates multiple visualizations—including pie charts, bar charts, and donut charts—to highlight top‑performing states, customer loyalty levels, and regional sales trends, offering a concise yet powerful tool for monitoring performance and guiding decision‑making.
